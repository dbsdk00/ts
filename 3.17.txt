1. 매개변수(function- return)
-  함수의 유연성을 높이고 재사용성을 높임
2. interface (객체 정의, 객체 상속)
- 코드 가독성, 유지보수성 높임
- 한 객체가 다른 객체를 상속
- 주로 사용되는 변수를 인터페이스로 지정하여 여러 곳에서 해당 구조를 재사용할 수 있음
- 속성을 선택적으로 적용할 수 있다 ('?' 사용)
3. type(const)
- 객체 타입뿐만 아닌 모든 유형의 타입을 정의할 수 있다

interface와 type은 비슷한 속성을 가짐
